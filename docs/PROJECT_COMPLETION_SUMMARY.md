## 🎯 **启动方式**

### 1. **标准启动**
```bash
cd /path/to/xianxia_world_engine
python run.py
```

### 2. **开发模式启动**
访问: `http://localhost:5001/intro?mode=dev`

### 3. **功能测试**

- 开发统计: `http://localhost:5001/dev/stats` (仅开发模式)
