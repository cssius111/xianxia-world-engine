## 🎮 如何开始游戏

### 快速开始
```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python main.py

# 或运行演示版本（有使用提示）
python play_demo.py
```

### 基础操作
1. **移动探索**
   - `地图` - 查看当前位置
   - `探索` - 探索当前区域
   - `去天南坊市` - 移动到其他区域

2. **角色管理**
   - `状态` - 查看角色信息
   - `技能` - 查看技能列表
   - `修炼` - 恢复并获得经验

3. **战斗操作**
   - `攻击 [目标]` - 普通攻击
   - `使用 [技能] [目标]` - 使用技能
   - `防御` - 防御姿态
   - `逃跑` - 逃离战斗

### 自然语言示例
系统支持更自然的表达方式：
- "我想去坊市看看"
- "帮我看看周围有什么"
- "我要修炼一会儿"
- "用剑气斩攻击敌人"

## 🛠️ 技术特点

### 模块化架构
```

├── core/          # 核心游戏系统
```
