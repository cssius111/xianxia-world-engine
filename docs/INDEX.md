# 📚 修仙世界引擎文档

## 🚀 快速开始
1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```
2. **配置环境**
   - 在 `.env` 中设置 `DEEPSEEK_API_KEY`
3. **运行项目**
   ```bash
   python run.py
   ```
